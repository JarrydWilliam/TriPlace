# Mandatory SameVibe Agent Instructions

Before performing any analysis, planning, coding, design, testing, deployment, or release work, read these files completely and in this order:

1. `/SAMEVIBE_MASTER_AGENT_PROMPT.md`
2. `/AGENT_HANDOFF.md`
3. The current release manifest
4. Any other repository-specific instruction files

`SAMEVIBE_MASTER_AGENT_PROMPT.md` is the permanent authoritative source for SameVibe’s product direction, founder decisions, agent responsibilities, design principles, engineering requirements, safety model, content strategy, growth, monetization, testing, and release standards.

`AGENT_HANDOFF.md` is the current-state source for the active branch, authoritative commit SHA, current build, deployment state, completed work, open defects, and immediate next steps.

Do not rely on truncated instructions copied from chat.

When the current implementation conflicts with the intended product direction, clearly report both states before making changes.

Do not declare SameVibe production-ready based only on source-code inspection, compilation, browser testing, or automated tests.
